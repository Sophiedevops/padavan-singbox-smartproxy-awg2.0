# padavan-singbox-smartproxy-awg2.0
Умный прокси-сервер на базе sing-box и AmneziaWG 1.5-2.0 для роутеров Padavan. Интерактивная гео-маршрутизация, точечный обход блокировок и минимальное потребление ресурсов (SOCKS5/HTTP/Shadowsocks)
# Padavan SmartProxy (sing-box + AmneziaWG)

Легковесный и умный автоматический установщик для настройки точечного обхода блокировок на роутерах со слабым железом (прошивка Padavan, чипы MT7620 и аналоги). Базируется на современном ядре `sing-box` и туннелях `AmneziaWG` с балансировщиком.

## 🌟 Особенности

* **Экономия CPU роутера:** Внутри домашней Wi-Fi сети трафик не шифруется. Роутер просто выступает как шлюз, что позволяет достичь максимальных скоростей без перегрузки процессора.
* **Умная гео-маршрутизация:** Встроенные профили для разных регионов:
  * **Профиль UA:** Трафик РФ (`geoip-ru`) принудительно заворачивается в туннель, остальной мир идет напрямую.
  * **Профиль RU:** Локальный трафик РФ идет напрямую провайдеру, а в туннель заворачиваются только выбранные сервисы.
* **Точечный обход (Split Tunneling):** Маршрутизация конкретных заблокированных баз (YouTube, Meta, Telegram, ChatGPT).
* **Kill QUIC:** Опциональная блокировка протокола QUIC (UDP 443) для форсирования стабильного TCP в приложении YouTube на Smart TV.
* **Универсальные входы:** * `1080` — HTTP Proxy (Идеально для Smart TV и Apple TV).
  * `1081-1083` — SOCKS5 (Для браузеров и Telegram).
  * `30183` — Shadowsocks (Формат ссылок `ss://` для мобильных клиентов V2rayNG / NekoBox).

## 📋 Требования

* Роутер с прошивкой Padavan (AsusRT, Prometheus и т.д.).
* Установленная среда Entware (смонтированная USB-флешка в `/opt`).
* Свободная ОЗУ (RAM): от 25 МБ.
* Свободное место на флешке: от 50 МБ.
